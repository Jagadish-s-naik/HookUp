# HookAI - Viral Content Generator

## Project Structure
- **/frontend**: React + Vite + Tailwind CSS application.
- **/backend**: Supabase configuration, Edge Functions, and database schema.

## Getting Started

### Frontend
1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Backend (Supabase)
The backend is managed via Supabase. Refer to `backend/supabase/schema.sql` for database setup.
To work with Edge Functions, install the Supabase CLI and link your project.
